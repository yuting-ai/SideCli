# SideCli v1.3.0 Release Notes / 更新说明

## 🇨🇳 中文版本 (Chinese)
此版本将原先基于 WebView 的 xterm.js 渲染层替换为了原生的 SwiftTerm 终端引擎。
这大幅度降低了系统资源开销，并带来了更加极致流畅、原生的终端交互体验。

---

## 🇺🇸 英文版本 (English)
This release replaces the WebView-based xterm.js rendering layer with the native SwiftTerm terminal engine.
This significantly reduces system resource overhead and delivers a much smoother, fully native terminal experience.
